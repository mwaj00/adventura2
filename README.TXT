Popis programu: 

Jedná se o jednoduchou hru typu advetura s textovým uživatelským rozhraním a s jednoduchou grafickým rozhraním.
 Hra obsahuje herní plán s osmi místnostmi. Hráč může přecházet z jedné místnosti do druhé. 

Hra se spouští v Netbeans tak, že vytvoříte instanci třídy TextoveRozhrani, dále voláte hraj(); nebo v příkazovém řádku: java -jar adventura.jar v příslušném adresáři.
Verze: 1.0
